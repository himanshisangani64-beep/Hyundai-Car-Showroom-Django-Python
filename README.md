# Hyundai Car Showroom Management System

## Project Description
The Hyundai Car Showroom Management System is a web application developed using Python and Django. 
It allows users to view car categories, browse products, add items to cart, and manage showroom data.

## Features
- User Registration and Login
- View Car Categories
- Browse Hyundai Car Models
- Add to Cart System
- Product Management
- Admin Dashboard
- Order and Billing System

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite Database

## Project Structure
- ProductApp – Manage car products
- CategoryApp – Manage car categories
- CartApp – Shopping cart system
- RegistrationApp – User registration and login
- DashboardApp – Admin dashboard
- BillApp – Billing system

## Installation Steps

1. Clone the repository
git clone https://github.com/himanshisangani64-beep/Hyundai-Car-Showroom-Django-Python.git


2. Go to project folder


cd Hyundai-Car-Showroom-Django-Python


3. Install dependencies


pip install -r requirements.txt


4. Run the server


python manage.py runserver


5. Open in browser


http://127.0.0.1:8000
